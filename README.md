# ToDoList-swiftui

Swift App. SwiftUI. To Do List. Core Data.

## How it works
* Create Your ToDo
* Choose ToDo's Category
* Add ToDo's Description
* Rearrange Sample ToDo's
* Delete Your ToDo
* Complete Your Todo
* Persist data with Core Data
